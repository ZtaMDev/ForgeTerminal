Features

Everything you need in a terminal

Split panes, tabs, web preview, session persistence, and more.

Split Panes

Create complex binary-tree terminal grids with horizontal and vertical splits. Drag-and-drop to rearrange or swap panes effortlessly.

Tabs & Sessions

Multi-tab interface with drag-to-reorder, pin/unpin, duplicate, and rename. Session restore remembers your exact layout across restarts.

Web Preview

Browse documentation or web content side-by-side with your terminal. Ctrl+Click links in the terminal to open them instantly.

Command Palette

Fuzzy-searchable palette (Ctrl+Shift+P) with every action at your fingertips. Navigate, split, rename, and manage your workspace without touching the mouse.

Beautiful & Customizable

Powered by the Catppuccin color palette. Customize your shell, font, cursor, animations, and more via the built-in settings panel.

Shortcuts

Keyboard-driven workflow

Every action has a shortcut. Forge uses a configurable Command Key (` by default) to avoid conflicts with your terminal applications.

Global

Ctrl + ` Toggle Passthrough Mode
Ctrl + Shift + ` New Terminal
Ctrl + Shift + P Command Palette
Ctrl + , Open Settings
Ctrl + Shift + O Past Sessions
Ctrl + Shift + Y Toggle Web Preview

Tabs

Ctrl + Tab Next Tab
Ctrl + Shift + Tab Previous Tab
Ctrl + 1 – 9 Jump to Tab
Ctrl + Shift + D Duplicate Tab
Ctrl + Shift + R Rename Tab

Split Panes

Ctrl + \ Split Horizontally
Ctrl + Shift + \ Split Vertically
Ctrl + W Close Current Pane
Ctrl + Shift + Arrow Cycle Splits
Ctrl + Alt + Arrow Swap / Move Pane

Terminal

Ctrl + C Copy
Ctrl + V Paste
Ctrl + Shift + F Search Terminal
Ctrl + = / - Zoom In / Out
